# Novias del Gato: Digital Ecosystem for Cali's Public Art

"Novias del Gato" is an interactive digital platform designed to manage and explore the famous cat sculptures of Santiago de Cali, Colombia. By combining geographic precision with community-driven data, the project ensures that both tourists and locals can reliably find, "capture," and learn about these iconic urban landmarks.

## Key Features
*   **Dynamic Mapping:** Real-time tracking of cat sculpture locations.
*   **Visual Check-ins:** Unlock cultural content by photographing sculptures on-site.
*   **Crowdsourced Accuracy:** A collaborative engine to report moved or missing sculptures.
*   **API-First Design:** Built for scalability and integration with city services.
